# 14-Day Tailwind CSS Course

A complete 14-day Tailwind CSS learning roadmap designed for beginners who want to build a strong foundation in modern frontend styling.

This repository provides a structured day-by-day curriculum with practical examples, reusable UI components, responsive layouts, and a final mini project.

---

## 📚 Course Structure

| Day    | Topic  |
|------  |--------|
| Day 01 | Tailwind CSS Introduction |
| Day 02 | Spacing & Sizing |
| Day 03 | Colors & Typography |
| Day 04 | Flexbox Layout |
| Day 05 | Grid System |
| Day 06 | Responsive Design |
| Day 07 | Buttons & Cards |
| Day 08 | Navbar Layout |
| Day 09 | Forms |
| Day 10 | Hover & Transition Effects |
| Day 11 | Position & Z-Index |
| Day 12 | Dark Mode |
| Day 13 | Component Building |
| Day 14 | Mini Project |

---

## 🚀 What You Will Learn

- Tailwind CSS fundamentals
- Utility-first workflow
- Responsive layouts
- Flexbox & Grid systems
- Typography and spacing
- Component creation
- Hover animations and transitions
- Dark mode implementation
- Building reusable UI structures
- Creating a complete landing page project
